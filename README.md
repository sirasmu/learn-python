# learn-python

I will be using WSL Ubuntu for this.

## Install Python

`sudo apt-get install python3`

## Hello World

1. Create and open a new python file with Vim: 

  `vim test.py`

2. Write the following program: 

       print("hello world :)")
  
3. Now you can run the program from the terminal with:

  `python3 test.py`
